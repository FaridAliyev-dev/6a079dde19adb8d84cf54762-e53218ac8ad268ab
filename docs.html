Introduction

Reis DGP (Data Governance Platform) is a modern metadata management and governance platform designed to help organizations discover, understand, monitor, and govern their data ecosystem at scale.

The platform centralizes metadata from databases, data warehouses, data lakes, BI tools, streaming systems, and transformation frameworks into a unified interface. Reis DGP enables engineering, analytics, governance, and business teams to gain visibility into their data assets, lineage, ownership, quality, and operational health.

Reis DGP is built for scalability, extensibility, and enterprise-grade governance, providing organizations with a reliable foundation for data discovery, observability, compliance, and metadata intelligence.

Core Capabilities

Reis DGP is designed with four primary pillars that address the most critical needs of modern data organizations.

Data Discovery

Reis DGP provides a centralized search and discovery experience for organizational data assets. Users can quickly locate datasets, dashboards, pipelines, tables, columns, machine learning assets, and documentation through an intelligent metadata index.

Search

Global metadata search across all assets.

Tagging

Asset tagging and classification.

Ownership

Granular ownership tracking.

  • Dataset browsing and schema visualization
  • Business glossary support
  • Popularity and usage insights
  • Semantic metadata relationships

Data Lineage

Reis DGP enables end-to-end lineage tracking across the data stack, helping teams understand how data flows between systems and transformations.

  • Upstream and downstream dependency mapping
  • Column-level lineage tracking
  • Pipeline relationship visualization
  • Impact analysis and root cause investigation
  • Transformation flow monitoring

Lineage visualization helps teams safely modify pipelines, investigate failures, and understand how business metrics are generated.

Data Governance

Reis DGP provides governance capabilities for enforcing standards, improving accountability, and maintaining trust across enterprise data systems.

  • Role-based access control (RBAC)
  • Dataset ownership management
  • Stewardship workflows and metadata policies
  • Data classification and compliance support
  • Audit logging and approval flows

Data Observability

Reis DGP includes observability features that help monitor metadata health, data quality, and operational reliability.

Platform Architecture

Reis DGP is designed using a modular and scalable architecture optimized for enterprise metadata workloads.

Frontend Layer

The frontend provides a responsive and user-friendly web interface for interacting with metadata assets and governance workflows.

  • Global metadata search
  • Interactive lineage visualization
  • Dataset exploration and documentation
  • Access management and observability monitoring

Backend Services

Backend services handle metadata processing, indexing, APIs, authentication, authorization, lineage computation, and ingestion orchestration.

Storage Layer

Reis DGP uses multiple storage technologies optimized for different metadata workloads.

  • Graph databases for lineage and relationships
  • Search indexes for fast discovery
  • Document stores for metadata persistence
  • Relational databases for operational data
  • Object storage for ingestion artifacts

Ingestion Framework

The ingestion framework is responsible for collecting, transforming, validating, and synchronizing metadata from external systems.

  • Scheduled ingestion jobs and real-time synchronization
  • Incremental metadata updates
  • Connector SDK support and schema extraction
  • Usage analytics ingestion

Search and Metadata Intelligence

Reis DGP includes an advanced metadata search engine capable of indexing structured and unstructured metadata.

  • Full-text metadata and column-level search
  • Faceted filtering and tag-based discovery
  • Popularity ranking and AI-assisted suggestions

Security

Security is integrated into every layer of Reis DGP. Single Sign-On (SSO), OAuth2, SAML, and granular RBAC ensure that your metadata remains protected.
